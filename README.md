# Learning System

A web-based learning system using:

- Django
- Tailwind
- Javascript/HTMX/Alpinejs

Create your own custom study guides, learning methods, exams and flashcards.

Track your progress.

## Exam Management

The system provides a comprehensive exam management system with the following components:

### Exams

Create and manage exam definitions with titles, descriptions, and active status.

### Test Banks

Organize questions into test banks within exams for better organization and targeted practice.

### Topics & Chapters

Structure exam content hierarchically using topics and chapters for logical content organization.

### Questions

Add multiple-choice questions with comprehensive explanations for each answer option.

### Acronym Flashcards

Study and memorize acronyms relevant to your exams with traditional flashcard functionality.

All exam components include creation timestamps and can be accessed through the Django admin interface.
